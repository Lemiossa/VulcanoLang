# VulcanoLang

Uma linguagem de programação simples feita em C

# Features
- operações basicas (<, >, <=, >=, ==, !=, +, -, *, /)
- pode concatenar strings com + e comparar com == / !=, não suporta <, >, <= ou >= ainda
- Suporte a IF e return
- Variáveis

# Basico

## expressões
```vulang
1 + 1 / 2;
```

## variáveis
```vulang
var x = 10;
```

## if
```vulang
if ( condição ) {
    ação
} else if ( condição ) {
    ação
} else {
    ação
}
```

atualmente só retorna o resultado da ultima expressão
